# Internationalization (i18n) — Django 6.0

## Overview

Django provides robust i18n and localization (l10n) for multi-language applications.

## Configuration

```python
# settings.py
from django.utils.translation import gettext_lazy as _

LANGUAGE_CODE = "en-us"

LANGUAGES = [
    ("en", _("English")),
    ("fr", _("French")),
    ("es", _("Spanish")),
    ("de", _("German")),
    ("ar", _("Arabic")),
]

USE_I18N = True  # Always True in Django 6.0
USE_TZ = True

LOCALE_PATHS = [
    BASE_DIR / "locale",
]

# Middleware
MIDDLEWARE = [
    "django.middleware.locale.LocaleMiddleware",  # After SessionMiddleware
    # ...
]
```

## Translation in Python Code

```python
from django.utils.translation import gettext, gettext_lazy, ngettext, ngettext_lazy

# Translate immediately
output = gettext("Welcome to our site")

# Lazy translation (for module-level strings)
from django.utils.translation import gettext_lazy as _

class Article(models.Model):
    title = models.CharField(_("title"), max_length=200)
    content = models.TextField(_("content"))

    class Meta:
        verbose_name = _("article")
        verbose_name_plural = _("articles")

# Pluralization
count = Article.objects.count()
message = ngettext(
    "There is %(count)d article.",
    "There are %(count)d articles.",
    count,
) % {"count": count}

# Lazy pluralization
message = ngettext_lazy(
    "There is %(count)d article.",
    "There are %(count)d articles.",
    count,
) % {"count": count}

# Contextual translation
from django.utils.translation import pgettext, pgettext_lazy

# "month" as in calendar month vs. month as in "30 days"
month = pgettext("calendar", "May")
month = pgettext("duration", "May")
```

## Translation in Templates

```html
{% load i18n %}

{# Basic translation #}
{% trans "Welcome to our site" %}
{% trans "Welcome" as welcome %}
<h1>{{ welcome }}</h1>

{# With context #}
{% trans "May" context "calendar" %}

{# Block translation (with variables) #}
{% blocktrans count counter=articles|length %}
    There is {{ counter }} article.
{% plural %}
    There are {{ counter }} articles.
{% endblocktrans %}

{# Block trans with variables #}
{% blocktrans with name=user.first_name %}
    Hello, {{ name }}!
{% endblocktrans %}

{# Block trans with context #}
{% blocktrans context "greeting" %}Hello!{% endblocktrans %}

{# String concatenation #}
{% blocktrans %}{{ title }} - {{ subtitle }}{% endblocktrans %}
```

## URL Patterns with i18n

```python
from django.conf.urls.i18n import i18n_patterns
from django.urls import path, include

urlpatterns = [
    path("api/", include("api.urls")),  # Not translated
]

urlpatterns += i18n_patterns(
    path("", include("main.urls")),
    path("articles/", include("articles.urls")),
    prefix_default_language=True,  # Include language prefix for default
)
# URLs: /en/articles/, /fr/articles/, /es/articles/
```

## Language Detection

LocaleMiddleware detects language in this order:
1. `request.session["django_language"]`
2. Language cookie (`django_language`)
3. `Accept-Language` HTTP header
4. `settings.LANGUAGE_CODE`

```python
# Manually set language
from django.utils import translation

translation.activate("fr")
# ... do work in French ...
translation.deactivate()  # Restore previous

# Get current language
current_lang = translation.get_language()
```

## Switch Language Template Tag

```html
{% load i18n %}

<form action="{% url 'set_language' %}" method="post">
    {% csrf_token %}
    <input name="next" type="hidden" value="{{ redirect_to }}">
    <select name="language" onchange="this.form.submit()">
        {% get_current_language as LANGUAGE_CODE %}
        {% get_available_languages as LANGUAGES %}
        {% get_language_info_list for LANGUAGES as languages %}
        {% for language in languages %}
            <option value="{{ language.code }}"
                {% if language.code == LANGUAGE_CODE %}selected{% endif %}>
                {{ language.name_local }} ({{ language.code }})
            </option>
        {% endfor %}
    </select>
</form>
```

## Making Translation Files

```bash
# Create/update message files
django-admin makemessages -l fr
django-admin makemessages -l es
django-admin makemessages -l de
django-admin makemessages -a  # All languages

# For JavaScript
django-admin makemessages -d djangojs -l fr

# Compile translations
django-admin compilemessages
```

### .po File Format

```po
# locale/fr/LC_MESSAGES/django.po
msgid "Welcome to our site"
msgstr "Bienvenue sur notre site"

msgid "There is %(count)d article."
msgid_plural "There are %(count)d articles."
msgstr[0] "Il y a %(count)d article."
msgstr[1] "Il y a %(count)d articles."

msgctxt "calendar"
msgid "May"
msgstr "Mai"
```

## Time Zones

```python
# settings.py
TIME_ZONE = "UTC"
USE_TZ = True

# Convert between timezones
from django.utils import timezone
from zoneinfo import ZoneInfo
import datetime

# Current time in UTC
now = timezone.now()

# Convert to specific timezone
tz = ZoneInfo("America/New_York")
ny_time = now.astimezone(tz)

# Local time
local_time = timezone.localtime(now)

# Make naive datetime aware
naive = datetime.datetime(2024, 1, 1, 12, 0)
aware = timezone.make_aware(naive, ZoneInfo("America/New_York"))

# Make aware datetime naive
naive = timezone.make_naive(aware, ZoneInfo("UTC"))
```

### Template Time Zone

```html
{% load tz %}

{# Convert to timezone #}
{% timezone "Europe/Paris" %}
    {{ article.published_at|date:"Y-m-d H:i" }}
{% endtimezone %}

{# Local time #}
{% localtime on %}
    {{ article.published_at|date:"Y-m-d H:i" }}
{% endlocaltime %}

{# Timezone filter #}
{{ article.published_at|timezone:"Europe/Paris" }}
```

## Localization Formatting

```python
from django.utils.formats import localize, date_format, number_format
from django.utils.translation import get_language

# Localize date
import datetime
date = datetime.date(2024, 1, 15)
formatted = date_format(date, "DATE_FORMAT")  # Language-specific format

# Localize number
number = 1234567.89
formatted = localize(number)  # "1,234,567.89" (en) or "1 234 567,89" (fr)

# Localize in template
{# {{ value|localize }} #}
{# {{ value|unlocalize }} #}
{# {{ value|localize }} #}
```
