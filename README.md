# Pledge Skills — Installable Agent Skill Library

[![skills.sh](https://skills.sh/b/pledgeandgrow/pledge-skills)](https://www.skills.sh/pledgeandgrow/pledge-skills)

A comprehensive, production-ready collection of 77 agent skills covering Next.js, React, React Native, TypeScript, Tailwind CSS, Astro, Bun, Express, Go, Python, Rust, Django, Kotlin, Vite, Redis, npm, PHP, Docker, GraphQL, Kubernetes, Vue.js, Svelte, Nuxt, Prisma, Spring, Jest, ESLint, Prettier, OpenTelemetry, SWC, Drizzle, Zustand, and more. Built from official documentation — every API, hook, component, type, utility class, and language feature.

## Quick Install

```bash
npx skills add pledgeandgrow/pledge-skills
```

Or browse on [skills.sh](https://www.skills.sh/pledgeandgrow/pledge-skills).

## Install

### Install all skills

```bash
npx skills add pledgeandgrow/pledge-skills
```

### Install specific skills

```bash
# Install individual skills
npx skills add pledgeandgrow/pledge-skills --skill nextjs
npx skills add pledgeandgrow/pledge-skills --skill react --skill typescript
npx skills add pledgeandgrow/pledge-skills --skill rust --skill zig --skill go

# Install all skills from the repo
npx skills add pledgeandgrow/pledge-skills --all

# Install to a specific agent (claude-code, cursor, codex, gemini-cli, etc.)
npx skills add pledgeandgrow/pledge-skills --skill nextjs -a claude-code

# Install globally (~/<agent>/skills/ instead of ./<agent>/skills/)
npx skills add pledgeandgrow/pledge-skills --skill nextjs -g

# Non-interactive (CI/CD friendly)
npx skills add pledgeandgrow/pledge-skills --skill nextjs -g -a claude-code -y
```

### Use a skill without installing

```bash
# Generate a prompt for one skill
npx skills use pledgeandgrow/pledge-skills --skill nextjs

# Start a coding agent interactively with the skill
npx skills use pledgeandgrow/pledge-skills --skill nextjs --agent claude-code
```

### List installed skills

```bash
npx skills list
# List only global skills
npx skills list -g
# Filter by agent
npx skills list -a claude-code -a cursor
```

### Find skills

```bash
# Interactive search
npx skills find
# Search by keyword
npx skills find typescript
# Search across an organization's repos
npx skills find react --owner pledgeandgrow
```

### Update skills

```bash
npx skills update
npx skills update nextjs
npx skills update -g -y
```

### Remove skills

```bash
npx skills remove nextjs
npx skills remove --all -y
```

### Manual install (copy folders)

Copy individual skill folders into your agent's skills directory (`.claude/skills/`, `.agents/skills/`, `.cursor/skills/`, etc.).
